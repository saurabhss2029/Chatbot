## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/saurabhss2029/Chatbot.git
cd Chatbot


# For frontend
cd frontend
npm install

# For backend
cd ../backend
npm install


# Start frontend
cd frontend
npm run dev

# Start backend
cd ../backend
npm start
